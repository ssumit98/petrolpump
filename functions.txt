PETROL PUMP MANAGEMENT SYSTEM - FUNCTIONAL CAPABILITIES

1. OWNER (ADMIN)
   - Dashboard Overview: View real-time tank levels, daily sales trends, and employee status.
   - Price Management: Update daily fuel rates (Petrol, Diesel, Power) which reflect instantly across the app.
   - Tank Stock Monitoring: Visual liquid gauge for tank levels.
   - Employee Oversight: Track employee shifts, status (Active/Inactive), and cash shortages.
   - Role Management: Create/Manage user roles (Manager, Pump Attendant, Customer).

2. MANAGER
   - Operations Dashboard (Daily Sheet):
     - Price Management:
       - Update Petrol/Diesel rates.
       - Auto-calculate Stock Gain/Loss impact based on current tank levels.
       - Generate and Download "Price Change Impact Report" (PDF).
     - Daily Sales Report:
       - Detailed Nozzle-wise breakdown (Opening/Closing Reading, Sales L, Amount â‚¹).
       - Aggregated Total Sales for Petrol and Diesel.
     - Financial Management:
       - Track "Payments Received" (Cash, Paytm, PhonePe, Credit).
       - Track "Expenses" (Shortage, Operational Expenses).
       - Auto-calculation of Net Collection (Payments - Expenses).
       - "Save Daily Sheet" to persist financial records.
   - Attendant Management:
     - Add New Attendants: Create accounts with Name, Auto ID, and Adhar Number.
     - Monitor Real-time "Cash in Hand" for each attendant.
     - View History: Calendar-based access to detailed shift history for any attendant.
   - Stock & Dip Management:
     - Record physical tank dip readings.
     - Auto-calculation of "Book Stock" (Opening Stock - Sales).
     - Variation Analysis: System highlights stock loss (Red) or gain (Green).
     - Update system stock levels.
    - Credit Management:
      - Log credit sales: Capture vehicle model and fuel type along with plate number.
      - Recent Transactions:
        - View history table for selected customer.
        - Search by vehicle number or amount.
        - Download PDF Statement.
      - Payment Verification:
        - View "Pending Payment Requests" from customers.
        - Approve Requests: Auto-logs "Payment" transaction and reduces customer balance.
        - Reject Requests: Updates status to "Rejected".
        - View uploaded receipts.
      - Real-time Credit Limit Validation: Blocks transactions if limit is exceeded.
      - Auto-updates customer outstanding balance.
      - Add New Customer: Create credit customer accounts with vehicle and credit limit details.

3. PUMP ATTENDANT (STAFF)
   - Mobile-First Entry Interface: Simplified, high-contrast UI for easy data entry on the forecourt.
   - Personalized Header: "Welcome, {Name}" with real-time "Cash in Hand" display.
   - Shift Management:
     - Start Job: Select nozzle, enter initial cash float. Previous cash in hand is auto-filled.
     - Active Job View: Large, clear display of current nozzle, start reading, and start time.
     - End Job: 
       - Enter End Reading (Total Litres auto-calculated).
       - Enter Cash Returned, Cash Online, and Change Given.
       - System auto-calculates Cash Remaining.
   - Cash Management:
     - Lend Cash: Send cash transfer requests to other attendants.
     - Accept Transfer: Securely accept cash from others. Funds are deducted from sender and added to receiver only upon acceptance.
     - Real-time Balance: Cash in Hand updates instantly after shifts or transfers.
   - History & Reporting:
     - Shift History Calendar: Calendar view of past work days.
     - Shift Details Popup: View detailed breakdown of any past shift (Time, Sales, Cash Handled, Change).
   - Voice Commands:
     - "Reading [number]": Auto-fills the End Reading input field.
     - "Logout": Voice-activated logout.
     - "Go Home": Navigation.

4. CUSTOMER
   - My Credits Portal: Personalized dashboard ("Welcome back, {Name}").
   - Outstanding Dues: View total amount to be paid and credit limit status.
   - Vehicle Management:
     - View Authorized Vehicles (Driver, Model, Fuel Type, Plate).
     - Add New Vehicle: Modal to add vehicle details including Fuel Type (Petrol/Diesel/CNG).
     - Search: Filter vehicles by Driver, Model, or Plate.
    - Payment Requests:
      - "Pay Now" Button: Initiate payment requests.
      - Optional Fields: Transaction ID and Receipt Upload are now optional.
      - Select Manager: Choose a manager for verification.
    - Transaction History:
      - Detailed table of credit purchases (Date, Vehicle Model/Fuel Type, Amount, Status).
     - Search: Filter transactions by Date, Vehicle, or Amount.
     - Download Statement: Generate PDF statement (button in table header).

5. GENERAL FEATURES
   - Role-Based Authentication: Secure login with automatic redirection to role-specific dashboards.
   - Voice Assistant: Global floating microphone button for voice navigation and commands.
   - Offline Mode: Visual indicator when internet connection is lost.
   - Responsive Design: Optimized for Desktop (Owner/Manager) and Mobile (Attendant/Customer).
